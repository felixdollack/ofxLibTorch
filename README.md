# ofxLibTorch

## Description
an openFrameworks wrapper for [LibTorch](https://pytorch.org/cppdocs/)

## Requirements
- openFramewrks v0.10.1
- LibTorch v1.1
- Windows10
- CUDA v10.0
- OpenCV

Also tested with:
- macOS Mojave (10.14.6)
- openFrameworks v0.11.0
- LibTorch v1.3.1

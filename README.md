# ofxLibTorch

## Description
an openFrameworks wrapper for [LibTorch](https://pytorch.org/cppdocs/)

## Requirements
- openFramewrks v0.10.1
- LibTorch v1.1
- Windows10
- CUDA v10.0
- OpenCV

Also tested on macOS with:
- LibTorch v1.3.1
- macOS Mojave (10.14.6)
